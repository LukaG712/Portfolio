Linux & Scripting

Installatie en configuratie van een Linux-server met Bash-scripts voor automatisering.

Over dit project

In dit project installeerde en configureerde ik een Linux-server (Ubuntu Server) en schreef ik meerdere Bash-scripts om systeemtaken te automatiseren. Ik werkte uitsluitend via de command line, zonder grafische interface.

Taken die ik automatiseerde: dagelijkse backups via cron-jobs, gebruikersbeheer, en monitoring van schijfgebruik met automatische e-mailwaarschuwingen bij overschrijding van een drempelwaarde.

Periode

Academiejaar 2025–2026

Technologieen

Linux Bash Cron SSH Ubuntu Server

Wat heb ik geleerd?

Linux werken via de terminal geeft je een dieper inzicht in hoe een besturingssysteem echt werkt. Ik leerde hoe bestandsrechten (chmod, chown) werken, hoe je gebruikers en groepen beheert, en hoe processen worden afgehandeld. Het systeem biedt veel controle als je weet hoe je ermee moet werken.

Bash-scripting leerde me denken in automatisering: wat doe je elke dag manueel, en hoe schrijf je een script dat dat voor jou doet? Variabelen, lussen, condities en functies in Bash zijn de bouwstenen van krachtige scripts.

Uitdagingen

De overgang van een grafische omgeving naar puur command line was in het begin een drempel. Elke actie vereist de juiste syntax en een typfout kan onverwachte gevolgen hebben. Ik leerde al snel het belang van backups en het testen van scripts in een veilige omgeving voor je ze uitvoert op een productieserver.
