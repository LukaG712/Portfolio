Eindproject

Integratie van alle geleerde vaardigheden in een volledig werkende webapplicatie.

Over dit project

Het eindproject is de kroon op de opleiding: een volledige webapplicatie die alle aangeleerde vaardigheden samenvoegt. De applicatie bestaat uit een responsieve front-end (HTML5, CSS3, Bootstrap), een back-end voor de businesslogica, een MySQL-databank voor dataopslag, en een netwerkcomponent voor communicatie tussen de lagen.

Het project werd uitgevoerd in teamverband en vereiste planning, taakverdeling en gebruik van versiebeheer via Git.

Periode

Academiejaar 2024–2025

Technologieen

HTML5 CSS3 Bootstrap PHP MySQL Git Linux

Wat heb ik geleerd?

Een eindproject dat alle vakgebieden combineert leerde me vooral hoe de verschillende stukken technologie samenwerken in een realistisch project. De front-end praat via HTTP-requests met de back-end, die queries uitvoert op de databank en resultaten terugstuurt als JSON. Dit volledige plaatje begrijpen en implementeren was een grote stap vooruit.

Samenwerken via Git met meerdere personen bracht nieuwe uitdagingen: merge-conflicten oplossen, branches beheren en duidelijke commit-messages schrijven zijn vaardigheden die je alleen leert door ze echt te doen.

Uitdagingen

De grootste uitdaging was de integratie: een bug in de back-end kan de front-end laten crashen, en een verkeerd SQL-schema maakt later aanpassingen pijnlijk. Goede planning en duidelijke afspraken over de API-structuur zijn cruciaal gebleken voor een soepel verlopend teamproject.
