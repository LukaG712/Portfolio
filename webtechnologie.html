Webtechnologie

Een responsieve portfolio-website gebouwd met HTML5, CSS3 en Bootstrap 5.

Over dit project

Voor dit project ontwikkelde ik een volledige responsieve portfolio-website met Bootstrap 5. De nadruk lag op semantische HTML5-structuur, correct gebruik van layout-elementen zoals header, main, footer, section en aside, en een strak, minimalistisch ontwerp dat werkt op elk schermformaat.

De website werd gedeployd via Netlify en is gelinkt aan een GitHub-repository voor versiebeheer. Het contactformulier maakt gebruik van Netlify Forms voor inzendingen zonder server-side code.

Periode

Academiejaar 2025–2026

Technologieen

HTML5 CSS3 Bootstrap 5 JavaScript Leaflet.js Netlify

Wat heb ik geleerd?

Dit project leerde me hoe ik een volledige website van nul opbouw met een professionele aanpak. Belangrijke inzichten waren het belang van semantische HTML voor toegankelijkheid en SEO, het denken in een responsief grid-systeem, en het gebruik van CSS-custom properties voor een consistent kleurenpalet door de hele website heen.

Ik leerde ook werken met Git en GitHub voor versiebeheer, en hoe je een statische site automatisch deployt via Netlify door simpelweg te pushen naar de main-branch. Het integreren van Leaflet.js op de contactpagina was een eerste kennismaking met een externe JavaScript-bibliotheek.

Uitdagingen

De grootste uitdaging was het consistent houden van de stijl over meerdere pagina's heen zonder code te herhalen. Door gebruik te maken van een gedeeld CSS-bestand en Bootstrap-utility classes kon ik dit oplossen. Het correct instellen van de Netlify Forms integratie vroeg ook wat uitzoekwerk, maar werkt uiteindelijk naadloos.
